Switch

switch is an alternative to if...else It's useful when you have a set of values that you need to check. 


      let grade = 'A';
      let gpaPoints = 0;
switch(grade) {
  case 'A':
    gpaPoints = 4;
    break;
  case 'B':
    gpaPoints = 3;
    break;
  default:
    gpaPoints = -1;
}
